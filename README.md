# Programacion_Web
Repositorio de la asignatura Programación Web donde se recopilan las prácticas y proyectos desarrollados durante el curso 2025/2026 en la Universidad de Granada. Cada carpeta contiene una práctica independiente con su propio código, recursos y documentación necesaria para su visualización.

En pe1 se ponen en práctica los conocimientos adquiridos sobre HTML5 y CSS3 durante el trascurso de la asignatura, con el objetivo de crear sitio web de una agencia de viajes, en el que se
mostrarán los diferentes viajes que se pueden reservar a través de la agencia y las diferentes opciones de alojamiento que se ofrecen en cada viaje. 
Los usuarios de la plataforma serán el administrador del sitio, que será el encargado de gestionar el sitio web de la agencia, y el público general que podrá reservar los viajes y hacer comentarios.

En pe2 se ponen en práctica los conocimientos adquiridos de PHP y JavaScript, así como la capacidad para incorporar otros conceptos no estudiados en las clases de prácticas. El contexto inicial será la web
estática que se desarrolló en pe1.
